## 功能

- **品牌浏览**：6 大品牌机身/镜头完整参数
- **系统评分**：画质/视频/便携/性价比/专业度 5 维评分
- **智能推荐**：输入预算 + 用途 → 自动推荐最优组合
- **组合评分**：任意机身+镜头组合评分
- **搜索**：全局快速查找

## 使用

直接用浏览器打开 `index.html` 即可。

## 推荐流程

1. 设定预算（¥1,000 - ¥100,000，支持全新/二手切换）
2. 设定照片/视频比例
3. 选择拍摄类型（1-3 项）
4. 选择画幅/品牌偏好（可选）
5. 输入已有设备（可选）
6. 查看推荐结果（Top 8 组合）

# 影视设备打分

🎬 相机机身 + 镜头组合评分系统

## 功能

- **5 大品牌**：佳能、尼康、富士、松下、索尼
- **机身数据库**：22 款主流相机，完整参数
- **镜头数据库**：22 款主流镜头，完整参数
- **组合评分**：机身 + 镜头系统综合评分（画质、视频、便携、性价比、专业度）
- **搜索**：快速查找设备
